Ecoshane Bank 商業模式九宮格

核心夥伴 (Key Partners)
  • 綠色金融機構（提供貸款產品）
  • 永續認證機構（驗證減碳成效）
  • 台灣電力公司（用電數據提供者）
  • API Gateway 服務供應商
核心活動 (Key Activities)
  • **範疇一與範疇二**碳排計算模型開發與維護
  • 開放銀行 API 介接與數據整合
  • AI Agent 演算模型訓練
  • 綠色貸款產品媒合與推薦
  • 網站與服務平台維護
價值主張 (Value Proposition)
  • **一站式淨零服務：** 整合碳排評估與金融媒合，簡化企業永續轉型流程。
  • **數據驅動決策：** 透過數據分析，提供客製化的減碳與金融方案。
  • **高效與精準：** 自動化數據收集與智慧化媒合，大幅節省時間與人力成本。
客戶關係 (Customer Relationships)
  • 平台自動化服務
  • AI 諮詢與客製化建議
  • 數據儀表板，提供即時回饋
  • 電子郵件與專人客服支援
通路 (Channels)
  • **數位平台：** 專題網站（GitHub Pages）、數據儀表板。
  • **行銷：** 內容行銷（部落格）、社群媒體、研討會與專案簡報。
  • **合作：** 透過金融夥伴與產業協會進行推廣。
客戶區隔 (Customer Segments)
  • **中小型企業 (SMEs)：** 缺乏資源進行內部碳盤查與永續轉型。
  • **碳密集型產業：** 面臨減碳壓力，尋求技術與金融支援。
  • **尋求綠色認證的企業：** 需精準數據來支持其永續報告與認證。
核心資源 (Key Resources)
  • **技術：** 碳排計算模型、AI 演算法、開放銀行 API 串接技術。
  • **數據：** **範疇一與範疇二**碳排放係數、用電量、燃料消耗、金融交易數據。
  • **人力：** 開發團隊、數據科學家、商業分析師。
成本結構 (Cost Structure)
  • **開發成本：** 程式開發、API 介接費用、軟體授權。
  • **運營成本：** 伺服器託管（AWS）費用、數據庫維護、API 服務費。
  • **推廣成本：** 行銷與業務推廣費用。
收益來源 (Revenue Streams)
  • **佣金模式：** 成功媒合綠色貸款後，向金融機構收取服務費。
  • **訂閱模式：** 企業用戶每月支付費用以使用碳排評估工具與 AI 建議。
